ResolveNow is a simple and efficient online complaint management system that allows users to register complaints, track their status, and communicate with administrators. This platform helps organizations streamline complaint handling and improve transparency and resolution efficiency.

🚀 Features

👤 User & Admin Login 📝 Register and Manage Complaints 📋 View and Track Complaint Status 📌 Admin Complaint Assignment System 💬 Messaging System between User and Admin 📊 Admin Dashboard for Complaint Management 📱 Responsive UI (Mobile + Web Friendly) 🔄 Real-Time Complaint Status Updates

🛠️ Tech Stack

Frontend: React.js, Bootstrap, JavaScript, HTML, CSS

Backend: Node.js, Express.js

Database: MongoDB

Authentication: JWT / Session-based Authentication

Deployment: Vercel / Netlify (Frontend) Render / Railway (Backend)

Installation & Setup Clone the repository git clone https://github.com/ShaikKouser22/Resolve-now---Your-Platform-for-Online-Complaints.git cd Resolve-now---Your-Platform-for-Online-Complaints

Install Backend dependencies cd backend npm install npm start

Backend runs on:http://localhost:8000

Install Frontend dependencies Open new terminal: cd frontend npm install npm start

Frontend runs on:http://localhost:3000

3000 🧪 Testing

Run frontend or backend tests using: npm test
